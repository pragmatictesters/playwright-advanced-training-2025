    What is HTTP Basic Authentication?

    HTTP Basic Authentication is a simple authentication scheme built into the HTTP protocol. When a server requires authentication, the browser displays a native login dialog (not an HTML form) prompting for username and password. The credentials are sent as a Base64-encoded header with each request.

    ‚ö†Ô∏è Key Insight for Playwright Testers:
    The browser's Basic Auth dialog is NOT part of the DOM ‚Äî you cannot use selectors like page.fill() or page.click() to interact with it. Dialog handlers like page.on('dialog') also do NOT work because Basic Auth prompts are handled at the browser/protocol level, not as JavaScript dialogs.

    üéì How to Handle Basic Auth in Playwright

    Method 1: httpCredentials in Config

    // playwright.config.ts
use: {
  httpCredentials: {
    username: 'admin',
    password: 'secret123'
  }
}

    ‚úÖ Best for: Project-wide authentication

    Method 2: browser.newContext()

    const context = await browser.newContext({
  httpCredentials: {
    username: 'admin',
    password: 'secret123'
  }
});

    ‚úÖ Best for: Test-specific credentials

    Method 3: Inline URL Credentials

    await page.goto(
  'https://admin:secret123@example.com/protected'
);

    ‚ö†Ô∏è Use with caution: Credentials visible in URL/logs
