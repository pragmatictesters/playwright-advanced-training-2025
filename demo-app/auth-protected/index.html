üí° For Playwright Tests

Use httpCredentials in your browser context or Playwright config to authenticate. Standard page selectors and dialog handlers will NOT work with HTTP Basic Auth prompts.

  • test.use({ httpCredentials: { username, password } })
  • browser.newContext({ httpCredentials: {...} })
  • Inline URL: https://user:pass@example.com

üåê Test with REAL Basic Auth (httpbin.org)

For real HTTP 401 authentication testing, use httpbin.org:

https://httpbin.org/basic-auth/admin/secret123

Playwright Example: 

test('real basic auth', async ({ browser }) => {
  const context = await browser.newContext({
    httpCredentials: {
      username: 'admin',
      password: 'secret123'
    }
  });
  const page = await context.newPage();
  await page.goto('https://httpbin.org/basic-auth/admin/secret123');

  // Returns: { "authenticated": true, "user": "admin" }
  await expect(page.locator('body')).toContainText('"authenticated": true');
});
